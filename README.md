# hazeBot
hazeBot adalah sebuah Bot Discord untuk para pemain game Genshin Impact.

## Arguments 
- !resin - perintah untuk melakukan timer pengisian ulang resin.
  - Example : !resin `<current resin>` `<desired resin>`

## Requirements
- Runtime `Python 3.8.10`
- Library `Discord.py`
